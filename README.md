# HobbyBook
To run cd in and "npm run dev"
HobbyBook is a React application that helps users discover learning resources for hobbies and creative projects. Users can search for a hobby, view book/resource results from the Open Library API, and save useful resources to their personal collection.

## Project Description

Hobbyists often need inspiration, guides, and learning resources, but searching across the internet can feel scattered. HobbyBook solves this problem by giving users one clean interface where they can search for hobby-related books and save useful resources for later.

The app is designed for makers, builders, creatives, and hobbyists who want to explore topics like woodworking, crochet, cars, gardening, photography, cooking, and more.

## User Problem

Many people want to learn new hobbies or improve existing skills, but finding helpful learning materials can take time. HobbyBook makes this easier by allowing users to search for books and resources based on a hobby or topic.

## Features

- Search for hobby-related books and resources
- View book titles, authors, publication years, topics, and cover images
- Save books to a personal saved list
- Remove books from saved resources
- Saved resources persist using localStorage
- Navigate between pages using React Router
- Loading, error, and empty-result states
- Responsive card-based layout

## API Used

This project uses the Open Library Search API.
https://openlibrary.org/search.json
